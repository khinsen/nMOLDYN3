Output

The results of an OPCM analysis are stored in a NetCDF file whose main variables are namely:
  • time: the times in ps at which all the time-dependent OPCM defined in Eq. 4.199 is computed,
and for each protein chain name found in the system
  • name_sequence: the residue numbers of the chain name,
  • name_s2: the time-dependent generalized order parameter defined for each residue of chain name as defined in Eq. 4.199,
  • name_s2_timeavg: the time-averaged generalized order parameter defined for each residue of chain name as defined in Eq. 4.200.
