Output

The results of a SD analysis are stored in a NetCDF file whose main variables are namely:
  • time: the times in ps at which all the time-dependant SFA is computed,
and for each protein chain name found in the protein
  • name_straightness_y: variables equals to the length of chain name - 4,
  • name_helixradius_y: variables equals to the length of chain name - 3,
  • name_orientdist_y: variables equals to the length of chain name - 1,
  • name_straightness: the straightness,
  • name_helixradius: the helix radius,
  • name_orientdist: the orientional distance.
