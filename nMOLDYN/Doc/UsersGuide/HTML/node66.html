Output

The results of a AC analysis are stored in a NetCDF file whose main variables are namely:
  • time: the times in ps at which the AC was evaluated,
  • triplet: the index for each triplet considered in the analysis,
  • ac_1_by_triplet: the angular correlation of for each triplet according Eq.4.34,
  • ac_2_by_triplet: the angular correlation of for each triplet according Eq.4.34,
  • ac_3_by_triplet: the angular correlation of for each triplet according Eq.4.34,
  • ac_1: the group-averaged angular correlation for according Eq.4.35,
  • ac_2: the group-averaged angular correlation for according Eq.4.35,
  • ac_3: the group-averaged angular correlation for according Eq.4.35.
