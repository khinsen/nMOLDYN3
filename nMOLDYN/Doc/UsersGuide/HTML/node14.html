nMOLDYN preference file

This file will be used by nMOLDYN to setup the preferences (e.g. documenation style, log file path ...). It is based on the ConfigParser module of the Python Standard Library [46] and as such it must respect the corresponding format. Section 4.1.6 will explain how to build, load or save such a file from nMOLDYN.
