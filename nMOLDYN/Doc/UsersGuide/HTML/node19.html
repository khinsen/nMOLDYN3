The File menu

Pressing the File menubutton brings up a menu from which it is possible to choose the following options:
  • Load NetCDF file
  • Trajectory conversion
  • Frame snapshot
  • Convert NetCDF to ASCII
  • Convert ASCII to NetCDF
  • Preferences
  • Quit
that will be described in the forthcoming sections.
