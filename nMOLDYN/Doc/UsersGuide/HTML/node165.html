Output

The results of a SD analysis are stored in a NetCDF file whose main variables are namely:
  • theta: the spherical angles in at which all the SD is computed,
  • phi: the spherical angles in at which all the SD is computed,
  • sd_rvalnm: the SD defined for the shell of radius r=val.
